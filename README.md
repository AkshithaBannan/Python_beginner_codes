# Python Beginner Codes

A collection of foundational Python scripts designed for beginners to learn core programming concepts such as loops, conditional statements, functions, string manipulation, and working with libraries.

---

## Included Programs

### 1. Prime Number Checker
Checks whether a given natural number greater than 1 is a prime number by verifying its divisors.
*   **Key Concepts:** `for` loops, `if-else` conditionals, `break` statements, nested `else` loops.

### 2. Product of Random Numbers
An interactive quiz game that generates two random integers between 1 and 100, calculates their product, and asks the user to guess the correct answer.
*   **Key Concepts:** `random` library (`randint`), user input handling, string concatenation, type casting (`int`, `str`).

### 3. Squares of Even Numbers
Automatically calculates and prints the squares of all even numbers within a specified range (configured for 100 to 200).
*   **Key Concepts:** `range()` with step parameters, arithmetic operators, loops.

### 4. Word Counter
Analyzes a string of text, cleans it of basic punctuation, converts it to lowercase, and counts the frequency of each individual word.
*   **Key Concepts:** String methods (`lower()`, `split()`, `rstrip()`), dictionaries, iterating through dictionary items.

### 5. Palindrome Checker
Determines whether a user-provided string reads the same backward as forward, ignoring case and non-alphanumeric characters.
*   **Key Concepts:** Functions (`def`), list comprehensions, string joining (`join`), string slicing (`[::-1]`).

---

## How to Run

1. Make sure you have **Python 3.x** installed.
2. Clone this repository or download the notebook file.
3. Open and run the code cells using a Jupyter Notebook environment, VS Code, or Google Colab.
